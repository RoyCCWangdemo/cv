The initial project assessment

After the initial consultation meeting but before accepting the actual project, I ask clients to fund a mini-project that acts as an initial assessment. This gives both myself and the client a chance to back out of the collaboration based the outcome of the assessment. By the end of this assessment, the client and I would identify the scope of my duties, have a few candidate work plans, and course-correction strategies should unexpected issues arise. After the initial assessment, we would start from the most promising work plan, and set up future assessments and milestones.

Scientific computing in Julia

Julia is a programming language that has syntax like MATLAB and R, but could approach the runtime speed of C. It is used by engineers, economists, and system biologists. It is now bundled with the latest Jupyter Lab. Julia natively interfaces with C, and Julia libraries can be called from Python and R. The converse is also true; I frequently call Python and R libraries from Julia in my projects.

I have been using Julia since 2013, and follow best coding practices and professional software design patterns that are accepted in the Julia development community. I have experience with writing Julia in combination with Python in Jupyter notebooks, Pluto notebooks (interactive notebooks), low-complexity web apps and REST APIs. I also do custom Julia/Python software library development, and wrapper library development in Python or R to a target Julia library.
